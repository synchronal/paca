Paca
====

Helpers for interacting with huggingface for llama.cpp.

This project is divided into two crates, `paca`, which defines data structures
and functions, and `paca-cli` for the CLI.

## CLI

Documentation on how to use the CLI may be found in [paca-cli's README](paca-cli/README.md).

## Installation

``` shell
brew install synchronal/tap/paca
```
